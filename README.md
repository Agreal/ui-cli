## Usage
1. clone this repo, `git clone git@github.com:Agreal/ui-cli.git`
2. `npm install`
3. `npm link` to add `ui` command to your ENV path
4. run `ui create <project>` to create a project
5. go into `<project>` and run `ui serve` to preview page
